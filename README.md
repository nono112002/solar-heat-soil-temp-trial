# 太陽熱養生 温度測定プロジェクト

## なぜやるか

太陽熱を利用した土壌養生（消毒）の効果を「感覚」ではなく「データ」で見たい。
養生中に実際にどれくらいの温度になっているのか、どのくらいの時間60℃以上を維持できているのかを記録し、農業×IoTの実装経験として記録に残す。

---

## 何を知りたいか（問い）

- 太陽熱養生中のハウス内・地中温度はどう推移するか
- 60℃以上の維持時間は養生効果の目安として使えるか
- Pico W + DS18B20 で安定したデータ取得・送信が実現できるか

---

## システム構成

```
ハウス内
  Pico W
    ├── DS18B20（地中温度）
    └── DS18B20（気温）
      ↓ MQTT（モバイルWiFi経由）
自宅ラズパイ → InfluxDB → Grafana → Hugo（ポートフォリオ）
```
