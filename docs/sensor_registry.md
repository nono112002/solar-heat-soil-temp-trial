# センサー台帳

最終更新: 2026-04-02

## 設計変更（2026-04-02）
1-Wireバス方式（GP7共有）から**個別GPIO方式**（GP8〜GP14）に変更。
IDスキャン・sensor_map.json廃止。GPIOピン番号がそのまま位置情報になる。
