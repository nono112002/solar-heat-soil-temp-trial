# PCB 設計仕様書 - PicoBox 基板

## 目的
DS18B20 センサーの配線煩雑さを解消するための専用基板。
Pico W + SDカード + 保護回路 + LED + スイッチ をひとつの基板にまとめる。

---

## GPIO ピン割り当て

| GPIO | 機能 | 接続先 |
|---|---|---|
| GP15 | TX LED（送信時点滅） | LED + 330Ω → GND |
| GP16 | SPI0 MISO | SDカード |
| GP17 | SPI0 CS | SDカード |
| GP18 | SPI0 SCK | SDカード |
| GP19 | SPI0 MOSI | SDカード |
| GP21 | 1-Wire BUS1 | DS18B20 中央地点（3本）+ 4.7kΩ → 3V3 |
| GP22 | 1-Wire BUS2 | DS18B20 エッジ地点（3本）+ 4.7kΩ → 3V3 |
| VSYS | 電源入力 | 5V（DC-DC出力から） |
| 3V3 | センサー電源 | DS18B20 VCC, LEDs |
| GND | グラウンド | 共通 |

---

## 回路構成

### 電源入力部
```
5V IN (2ピン端子台)
  → ポリスイッチ 500mA（自動復帰ヒューズ）
  → スライドスイッチ（ON/OFF）
  → 逆接防止ショットキーダイオード（1N5819等）
  → VSYS（Pico W ピン39）
  → 電源LED: 3V3 → 330Ω → LED(緑) → GND  ※Picoが動作中=点灯
```

### DS18B20 接続部（BUS1 / BUS2 各1系統）
```
3V3 → 4.7kΩ → GPxx（データライン）
                ↓
DS18B20コネクタ × 3個（3ピン端子台: VCC / DATA / GND）
※3本がバス共有のため DATAは全て同じピンに接続
```

### TX LED
```
GP15 → 330Ω → LED(赤or青) → GND
```

### SDカードモジュール
```
秋月 105488 DIP化キット をピンヘッダで接続
（基板上にソケット実装）
```

## 基板サイズ・外形

- 目安: 80×60mm（防水ボックス 220×170×110mm に収まるサイズ）
- Pico W はソケット実装（着脱可能）
- 基板3枚同一設計（区A/B/C 共通）
