# PCB 設計仕様書 - PicoBox 基板

## システム構成概要

```
【上流電源ボックス】（1台）
  24V入力 → DC-DC 5V変換 ─[D1: 逆接防止]──┐
  電池ボックス(単3×3本=4.5V) ─[D2: Schottky]─┴──→ 5V バス出力
                                                   ├──→ PicoBox A（区A）
                                                   ├──→ PicoBox B（区B）
                                                   └──→ PicoBox C（区C）

【PicoBox】（3台、同一設計）
  5V バス入力 → ポリスイッチ → Pico W + SDカード + センサー端子台
```

---

## 上流電源ボックス 回路

### ダイオードOR回路（バックアップ切り替え）

```
24V → DC-DC 5V converter ─[D1: 1N5819（逆接防止のみ）]──┐
                                                          ├──→ 5V バス → 各PicoBoxへ
電池ボックス(単3×3本=4.5V) ──────[D2: Schottky 1N5819]──┘
```

**動作原理**:
- 主電源ON: バス = 5V、電池4.5V < 5V のためD2が逆バイアス（電池から電流流れない）
- 主電源OFF: 電池→D2→4.2V がバスに供給、3台のPicoへ無停止で切り替わる

**電池バックアップ持続時間（参考）**:
| 状態 | 3台合計電流 | 単3×3本（2500mAh）持続時間 |
|---|---|---|
| 平均（30分に1回送信） | ~30mA | ~80時間（約3日） |
| 待機のみ | ~15mA | ~170時間（約7日） |

## PicoBox 回路

### 外部接続（フェルール端子 + WAGO/Phoenix）

| 端子 | 信号 | 備考 |
|---|---|---|
| 5V IN / GND | 電源入力（上流ボックスから） | 2ピン端子台 |
| DS18B20 × 7本 | VCC / DATA / GND | 3ピン端子台 × 7 |

### GPIO ピン割り当て

| GPIO | 機能 | 接続先 |
|---|---|---|
| GP2  | SPI0 SCK  | SDカード |
| GP3  | SPI0 MOSI | SDカード |
| GP4  | SPI0 MISO | SDカード |
| GP5  | SPI0 CS   | SDカード |
| GP6  | SD カード検出（挿入時 LOW） | SDモジュール CD ピン + PULL_UP |
| GP7  | 1-Wire BUS（全センサー共通） | DS18B20 最大7本 + 4.7kΩ → 3V3 |
| GP15 | TX LED（送信時点滅） | LED + 330Ω → GND |
| GP_MON（未定） | 主電源監視 | 分圧回路（10kΩ/10kΩ）→ GPIO |
| VSYS | 電源入力 | 5V バスから |
| GND  | グラウンド | 共通 |

> **変更履歴**: GP16-19 → GP2-5（2026-03-29）/ GP20 → GP6（SD card detect）

### 電源入力部

```
5V バス入力
  → ポリスイッチ 500mA（自動復帰ヒューズ）
  → スライドスイッチ（ON/OFF）
  → VSYS（Pico W）
  → MCP1703-3302（外部LDO）→ 3.3V（SD・センサー・LED用）
```

### 3.3V レギュレーター（外部LDO）

**MCP1703-3302（SOT-23）**
- ドロップアウト電圧: 250mV（電池バックアップ時 4.2V入力でも3.3V出力OK）
- AMS1117-3.3は不可（ドロップアウト1.5V、4.2V入力では不足）

```
5V バス → MCP1703-3302 → 3.3V
            入力: 100nF セラコン
            出力: 10µF セラコン
```

PicoのWiFi（内部3.3V）とSD・センサーの3.3Vが完全分離 → 電圧干渉なし。

### 主電源監視回路（電源断アラート）

```
5V バス ─[R1: 10kΩ]─┬─[R2: 10kΩ]─ GND
                     └──→ GP_MON（約2.5V → Pico が HIGH 認識）
```

- 主電源あり → HIGH → 通常動作
- 主電源なし → LOW → ファームウェアがMQTTアラート送信（電池残量中に送出）

### DS18B20 接続部（個別GPIO方式）

各センサーを専用GPIOに1対1接続。IDスキャン・sensor_map不要。
1本が故障してもバス全滅しない。交換もピンに挿し直すだけ。

```
+3V3_EXT → 4.7kΩ → GP8  → DS18B20 S1（中央 10cm）
+3V3_EXT → 4.7kΩ → GP9  → DS18B20 S2（中央 25cm）
+3V3_EXT → 4.7kΩ → GP10 → DS18B20 S3（中央 40cm）
+3V3_EXT → 4.7kΩ → GP11 → DS18B20 S4（エッジ 10cm）
+3V3_EXT → 4.7kΩ → GP12 → DS18B20 S5（エッジ 25cm）
+3V3_EXT → 4.7kΩ → GP13 → DS18B20 S6（エッジ 40cm）
+3V3_EXT → 4.7kΩ → GP14 → DS18B20 S7（外気温）
```

各ラインに 100Ω 直列抵抗（GPIO保護）を追加。
プルアップ 4.7kΩ × 7本、直列 100Ω × 7本。

### SDカードモジュール（AE-MICRO-SD-DIP / 秋月電子）

**モジュール品番**: AE-MICRO-SD-DIP（秋月電子）
**接続モード**: SPI（4線式）
**KiCADコネクタ**: J9 = Conn_01x10（10ピン）

#### モジュールピン配置とSPIマッピング

| モジュールPin | 信号名 | SPIでの役割 | 接続先 | ネット名 |
|---|---|---|---|---|
| ① | DAT2 | 未使用（SPI不要） | NC | - |
| ② | CD/DAT3 | CS（チップセレクト） | GP5 | SD_CS |
| ③ | CMD | MOSI | GP3 | SD_MOSI |
| ④ | VDD | 電源 | +3V3_EXT | +3V3_EXT |
| ⑤ | CLK | SCK | GP2 | SD_SCK |
| ⑥ | VSS | GND | GND | GND |
| ⑦ | DAT0 | MISO | GP4 | SD_MISO |
| ⑧ | DAT1 | 未使用（SPI不要） | NC | - |
| ⑨ | カード検出スイッチB | CD（GPIO入力側） | GP6 | SD_CD |
| ⑩ | カード検出スイッチA | CD（GND側） | GND | GND |

> **カード検出スイッチ**: カード挿入時にスイッチが閉じ、⑨-⑩間が短絡。GP6をPULL_UP設定にすると、カード挿入時にLOW、未挿入時にHIGHとなる。
> **DAT1・DAT2**: SPIモードでは使用しない。NC（未接続）で問題なし（SD内部にウィークプルアップあり）。

#### Pico W GPIO割り当て（物理ピン4〜10）

| 物理ピン | GPIO | 機能 |
|---|---|---|
| 4 | GP2 | SCK  |
| 5 | GP3 | MOSI |
| 6 | GP4 | MISO |
| 7 | GP5 | CS   |
| 8 | GND | GND  |
| 9 | GP6 | CD（カード検出） |
| 10 | GP7 | 1-Wire（DS18B20全センサー共通） |

**7ピン連続（物理ピン4〜10）で全信号線が片側に集約。**

### DS18B20 ライン保護

```
GP8〜GP14 各ピン → 100Ω直列抵抗 → センサーコネクタ DATA
理由: 静電気・誤配線時の GPIO ピン保護（7本分）
```

## 電源設計 注意事項（実機確認より）

**症状**: SD カードマウント失敗 / "timeout waiting for v2 card"

**原因**:
- Pico W の CYW43 WiFi チップが内部3V3レールを共有
- WiFi 動作中は 3V3 が 2.9V 以下に降下（**実測 2026-03-29**）
- Freenove 拡張ボードの 3V3 端子台の**ねじが緩んでいた**ことも一因

**PCBでの根本対策**:
- SD・センサーの3.3VはMCP1703-3302から供給 → WiFi干渉と完全分離
- ファームウェアの`wifi_off()`→SDマウントシーケンスは引き続き維持
- 教訓: **ねじ端子台は必ず締め付けトルクを確認すること**
