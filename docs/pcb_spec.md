# PCB 設計仕様書 - PicoBox 基板

## 目的
DS18B20 センサーの配線煩雑さを解消するための専用基板。
Pico W + SDカード + 保護回路 + LED + スイッチ をひとつの基板にまとめる。

---

## GPIO ピン割り当て

| GPIO | 機能 | 接続先 |
|---|---|---|
| GP2  | SPI0 SCK  | SDカード |
| GP3  | SPI0 MOSI | SDカード |
| GP4  | SPI0 MISO | SDカード |
| GP5  | SPI0 CS   | SDカード |
| GP15 | TX LED（送信時点滅） | LED + 330Ω → GND |
| GP20 | SD カード検出（挿入時 LOW） | SDモジュール CD ピン + PULL_UP |
| GP28 | 1-Wire BUS（全センサー共通） | DS18B20 最大7本 + 4.7kΩ → 3V3 |
| VSYS | 電源入力 | 5V（DC-DC出力から） |
| 3V3  | センサー電源 | DS18B20 VCC, LEDs |
| GND  | グラウンド | 共通 |

> **変更履歴**: GP16-19 → GP2-5 に変更（2026-03-29 ブレッドボード実機確認済み）

## 回路構成

### 電源入力部
```
5V IN (2ピン端子台)
  → ポリスイッチ 500mA（自動復帰ヒューズ）
  → スライドスイッチ（ON/OFF）
  → 逆接防止ショットキーダイオード（1N5819等）
  → VSYS（Pico W ピン39）
  → 電源LED: 3V3 → 330Ω → LED(緑) → GND  ※Picoが動作中=点灯
```

### DS18B20 接続部（GP28 単一バス）
```
3V3 → 4.7kΩ → GP28（データライン）
                ↓
DS18B20コネクタ × 7個（3ピン端子台: VCC / DATA / GND）
※全本バス共有のため DATA は全て GP28 に接続
```

### TX LED
```
GP15 → 330Ω → LED(赤or青) → GND
```

### SDカードモジュール
```
秋月 105488 DIP化キット をピンヘッダで接続
（基板上にソケット実装）
```

### SDカード電源デカップリング（必須）
```
3V3 → 100nF（セラコン）→ GND  ※SDモジュール VDD 直近に実装
理由: SDカード初期化時に瞬間的な電流ピークが発生する。
     Pico W の CYW43 WiFi チップも同じ 3V3 レールを共有しており、
     WiFi動作中は電圧が2.9V以下まで降下する場合がある（実測値）。
     → WiFi OFF 後に SD マウントするシーケンスが必須。
     → コンデンサでも保護しきれない場合は外部電源レギュレータを使用。
```

---

## 電源設計 注意事項（実機確認より）

### 問題: Pico W 内蔵 3V3 レールの電流供給能力不足

**症状**: SD カードマウント失敗 / "timeout waiting for v2 card" / "no SD card"

**原因**:
- Pico W の CYW43 WiFi チップが 3V3 レールを共有
- WiFi 動作中は 3V3 が 2.9V 以下に降下（**実測 2026-03-29: ブレッドボードで 2.9V 確認**）
- SD カードは電源投入時に ~100mA の突入電流が発生

**2026-03-29 実機トラブルシューティング記録**:
- Freenove 拡張ボードの 3V3 端子台の**ねじが緩んでいた**ことが電圧降下の主因のひとつ
- ブレッドボード実装時は接触抵抗が加わり 2.9V まで低下
- ジャンパワイヤ直結 + WiFi OFF シーケンスで解消
- **教訓: ねじ端子台は必ず締め付けトルクを確認すること**

**対策（必須）**:
1. **シーケンス制御**: `wifi_off()` を実行してから SD マウント（firmware 実装済み）
2. **デカップリングコンデンサ**: SDモジュール VDD 直近に 100nF セラコン
3. **バルクコンデンサ**: 電源ライン全体に 100µF 電解コンデンサ

**根本対策（PCB設計推奨）**:
- Pico W の 3V3 レール（最大200mA）はセンサー・LED専用に限定
- SD カードは独立した外部レギュレータ（例: AMS1117-3.3）から給電を検討
- PCB ではすべての電源ピンにスルーホール半田付けで接触抵抗を排除

## サージ・保護回路

### 入力サージ対策
```
5V IN → TVSダイオード（5.0V, 単方向）→ GND
理由: 長いケーブル配線での誘導サージ、電源ON/OFF時のスパイク対策
推奨部品: P6KE6.8A または SMBJ5.0A（SMD）
```

### DS18B20 ライン保護
```
GP28 → 100Ω直列抵抗 → 1-Wire バス
理由: 静電気・誤配線時の GPIO ピン保護
（プルアップ 4.7kΩ は既存）
```

---

## 外部電源構成（Pico W の 3V3 に頼らない場合）

```
外部 5V 電源（ACアダプタ等）
  → ポリスイッチ → スライドスイッチ → ショットキーダイオード（逆接防止）
  → VSYS（Pico W）          ← Pico 内部レギュレータで 3.3V 生成（WiFi・MCU用）
  → AMS1117-3.3 等の外部LDO ← SD カード・センサー専用 3.3V レール
```

**分離のポイント**: WiFi/MCU と SD/センサーの 3V3 レールを分けることで電圧干渉を排除。
